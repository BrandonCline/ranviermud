RanvierMUD

Ranvier is a Node.js based MUD engine built for modularity, extensibility and getting up-and-running in the least amount of time possible. Full localization support. Mob/item/room scripting right in Javascript, no more terrible DSL. And most importantly, thorough documentation.

Quickstart

Installing Dependencies Node.js >= 0.6.2 

git clone git://github.com/shawncplus/ranviermud
cd ranviermud
npm install
sudo ./ranvier -v --save=10 --respawn=10

Download

You can download this project in either zip or tar formats.

You can also clone the project with Git by running: 

$ git clone git://github.com/shawncplus/ranviermud
