Features Stuff that you get out of the box

Localization
Support for localizing any string displayed to the player after they log in. E.g., Player A and Player B log in, play the same game and see a different language. Same items, npcs, rooms, etc, different language.
Scripting
It's in Javascript! No need for a shitty DSL. The codebase is javascript, the scripting is javascript.
Events
Scripting is event-based. Put a listener in the script attached to your mob and it'll automagically happen when the event is emitted. Adding new events is easy as npc.emit('eventName');
JSON Playerfile Storage
Pfile saving in JSON, easy to parse for external usage
YAML-described Entities
Rooms, items, and npcs are easily described in yaml. No wierd ~~~~ and byte-specific parsing that requires a GUI editor to build.
Flexible Room System
Arbitrary room exits custom to each room not locked into n/s/e/w and it's not a pain in the ass to add them
Leveling
Built into the codebase is a basic leveling system that can be easily customized to your desired xp formulae
Combat
Due to the way the repl works it could be used to do turn-based or real-time combat without changing the base functionality.
Skills
Basic, easily customizable/overhauled skill system
Channels
Basic channel system and incredibly simple framework for adding new channels
